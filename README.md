<span id="token">

### Clonar repositorio con Personal Token (PT)

~~~

git clone https://[USUARIO]:[PT]@github.com/[USUARIO]/[REPO]

~~~

<span id="add">

### Añadir cambios localmente

~~~

git add .

~~~

<span id="commit">

### Commitear cambios

~~~

git commit -m "Comentario"

~~~

<span id="push">

### Subir cambios

~~~

git push

~~~

<span id="pull">

### Descargar cambios

~~~

git pull

~~~

<span id="log">

### Mostrar logs de los push realizados con su ID

~~~

git log --oneline

~~~
